Como usar IKUSA: web, linea de comandos, MCP y API REST

    


    

      

        
1 Uso web

        
Arrastra un .apk o .aab a la zona de subida
          (max 150 MB), elige el tipo de analisis y pulsa "Analizar APK". El
          analisis corre en segundo plano: no cierres la pestana.

        
    
          
  • SAST sin LLM — solo analisis estatico MobSF. El mas rapido.
    • 
          
  • SAST + IA (LLM) — anade triage con IA: prioridad y explicacion en lenguaje claro.
    • 
          
  • MASVS/CRA Report — lo anterior mas el PDF de compliance descargable.
    • 
        

        
El triage con IA tarda ~2-3 minutos en
          CPU (segundos con GPU). El CRA Readiness Score
          va de 0 a 100: cuanto mas alto, mas cerca de cumplir.

      


      

        
2 CLI

        
Para integrar en CI/CD. Devuelve codigo de salida 2 si
          hay findings por encima del umbral indicado.

        
uv run ikusa-cli login --api-key ikusa_sk_demo
uv run ikusa-cli scan app.apk -o report.pdf --fail-on alto
# Exit 0 = ok, 2 = findings por encima del umbral

      


      

        
3 MCP (asistentes de IA)

        
Servidor JSON-RPC sobre stdio para clientes compatibles con MCP.
          Expone 4 tools: scan_apk, get_scan_status,
          get_scan_findings, list_scans.

        
uv run ikusa-mcp

        
Registro en el cliente MCP (Cursor, etc.):

        
{
  "mcpServers": {
    "ikusa": {
      "command": "uv",
      "args": ["run", "ikusa-mcp"],
      "cwd": "/ruta/a/ikusa-prototype"
    }
  }
}

      


      

        
4 API REST

        
POST/scansube APK (campo file) + tier; devuelve scan_id

        
GET/scan/{id}estado del scan (status / stage / message)

        
GET/scan/{id}/resultfindings + cra_score en JSON

        
GET/scan/{id}/reportPDF de compliance

        
GET/scanshistorial del usuario

        
GET/accounttier y cuota

        
Ejemplo:

        
curl -F file=@app.apk -F tier=compliance \
     -H "Authorization: ikusa_sk_demo" \
     http://localhost:9787/scan

        
Autenticacion opcional con la cabecera
          Authorization: ikusa_sk_*. Sin clave se usa el tier
          free anonimo.
